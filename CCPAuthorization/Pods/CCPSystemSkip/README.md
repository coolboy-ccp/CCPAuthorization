# CCPSystemSkip


## iOS常见内置App跳转(iOS 10.0+)

### How to Use 
```ruby
pod 'CCPSystemSkip'
```
import SystemSkip

eg. use `SystemSkip.wifi.skip()`  skip to wifi setting.
